# ProofStamp Documentation

Welcome to the ProofStamp documentation hub! This directory contains detailed guides on the architecture, legal compliance, and setup instructions for the project.

- 🏗️ **[Technical Architecture](TECHNICAL_ARCHITECTURE.md)**: A deep dive into the system architecture, authentication flows, and component structure.
- ⚖️ **[Indian Legal Compliance](INDIAN_LEGAL_COMPLIANCE.md)**: Detailed mapping of ProofStamp's features against the Bharatiya Sakshya Adhiniyam, 2023 (BSA) and the Information Technology Act.
- 🛡️ **[AI Protection Guide](AI_PROTECTION_GUIDE.md)**: Explains the DWT-DCT invisible watermarking and perceptual hashing mechanisms used to protect creators from unauthorized AI training.
- 🔍 **[Verification & Legal Guide](VERIFICATION_AND_LEGAL_GUIDE.md)**: How to verify stamps, check the blockchain anchors, and validate RFC 3161 timestamps.
